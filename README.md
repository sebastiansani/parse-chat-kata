# parse-chat-kata

## Requirements
* Python 3

## How to run tests:
```
git clone https://github.com/sebastiansani/parse-chat-kata.git
cd parse-chat-kata/
python test_chat.py
```

## How to run code from terminal:
```
git clone https://github.com/sebastiansani/parse-chat-kata.git
cd parse-chat-kata/
python chat.py "14:24:32 Customer : Lorem ipsum dolor sit amet, consectetur adipiscing elit."
```
